# Weather Data Dashboard Project

This project analyzes synthetic weather data (2015–2025) for 6 Indian cities using **Matplotlib** and **Plotly**.

## Files
- `weather_data.csv` : Synthetic dataset with columns (Year, Month, City, Temperature, Rainfall, Humidity)
- `Weather_Dashboard.ipynb` : Jupyter Notebook with code cells (analysis & visualizations)
- `README.md` : Instructions

## Requirements
```
pandas
numpy
matplotlib
plotly
nbformat
```

## How to Run
1. Extract the ZIP file.
2. Open Jupyter Notebook and load `Weather_Dashboard.ipynb`.
3. Run cells in order.
4. Use interactive Plotly visualizations in the notebook.

## Notes
- Data is synthetic. Replace `weather_data.csv` with real weather data (from APIs or IMD datasets).
- Extend project with interactive dashboards (Streamlit / Dash).
